# Agnitra

**Agnitra is the trust and optimization layer for production LLM inference — faster, cheaper, cryptographically signed.** One Python keyword. No retraining. Drop-in replacement for `model.generate()` calls.

```python
from agnitra.integrations.huggingface import AgnitraModel

model = AgnitraModel.from_pretrained(
    "meta-llama/Meta-Llama-3-8B-Instruct",
    torch_dtype=torch.float16,
    agnitra_kwargs={"input_shape": (1, 512), "quantize": "int8_weight"},
).cuda()
```

That's it. No retraining. No graph rewrites by hand. No serving stack to adopt.

## What Agnitra optimizes

Agnitra is intentionally narrow. The wedge is **decoder-only LLMs** — Llama-class models that account for ~80% of LLM inference spend in production. Every fine-tune of every supported architecture inherits the optimization decisions of its base model via [architecture fingerprinting](/docs/intro/architecture), so "13 architectures supported" effectively means "the ~100K decoder-LM fine-tunes on Hugging Face."

See [Supported architectures](/docs/reference/configuration#supported-architectures) for the full list.

## Get started in 60 seconds

- **Install:** `pip install agnitra` (add `[quantize]` for INT8 support)
- **Run** the [Quickstart](/docs/intro/quickstart) — works on any decoder-LM you can load with `transformers`.
- **Integrate** with your existing stack: [HuggingFace](/docs/guides/cli-and-sdk#huggingface), [LangChain](/docs/guides/cli-and-sdk#langchain), [LlamaIndex](/docs/guides/cli-and-sdk#llamaindex), or [accelerate](/docs/guides/cli-and-sdk#accelerate).

## Two things to know up front

1. **`torch.compile` defaults already include SDPA + static cache.** Plain Agnitra (without quantization) matches `torch.compile` on Llama-3-8B in `transformers 4.44.2` rather than beating it. **The actual speedup lever is `quantize="int8_weight"`** — see [INT8 quantization](/docs/guides/cli-and-sdk#int8-quantization).
2. **Honest scoping.** Models outside the supported architecture set get a passthrough `RuntimeOptimizationResult` with `notes["passthrough"]=True`. Agnitra never silently no-ops — you always know whether the optimizer ran.
